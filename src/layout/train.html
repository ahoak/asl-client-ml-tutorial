Basic Process

  1. Use the Mediapipe model to extract the set of joint positions of each image in the dataset.
  2. Flatten the joint positions into an array of values with the format: [joint1_x, joint1_y, joint1_z, ..., jointN_x, jointN_y, jointN_z]
  3. Normalize the flattened joint positions so that their values are scaled from -1 to 1.
  4. Shuffle the data.
  5. Use one-hot encoding to transform letter labels (i.e. 'A', 'B', 'C') to a machine-friendly data format.
  6. Split data into a training set and validation set.
